# AppWidgetDemoG
CalendarWidget 日历桌面挂件

 * 显示今天时间（阳历、阴历）  
 * 添加挂件时可设置挂件背景样式  
 * 可添加特殊节日（只记录农历月、日）  
 * 特殊节日到期通知提醒，突出颜色  
 * 挂件每30分钟刷新一次  
   
 * 电话号注册（mob手机号验证）  
 * 根据电话生成自己得数据文件  
 * 数据文件加密后保存在七牛云<对象存储>  
 </br>
